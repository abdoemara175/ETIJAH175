# ETIJAH175 - Focus-first Behavioral Control Interface (V2)

## نظام التحكم السلوكي الموجه بالتركيز

**النظام يقرر. المستخدم ينفذ. بدون تفاوض مع التشتت.**

---

## نظرة عامة

ETIJAH175 هو نظام تشغيل سلوكي يحول النية إلى إجراء فوري. يتم تصميمه لاستئصال التردد وفرض التنفيذ من خلال:

- **محرك قرار ديناميكي**: يختار الإجراء أو المهمة الأمثل بناءً على حالتك الحالية
- **نظام مضاد للإدمان**: يكتشف أنماط التشتت والإدمان ويتدخل فوراً
- **نظام التحقق من التنفيذ**: يتطلب تأكيداً لكل مخرجات (تم/تخطي/فشل)
- **الذاكرة المحلية الدائمة**: يتتبع السجل والشريط والأنماط

---

## المميزات الرئيسية

### 1. **نظام الحالة السلوكية**

يصنف النظام حالتك قبل الاختيار:

- **FOCUSED** (مركز): تعيين مهام عميقة
- **DISTRACTED** (مشتت): إجراءات مضادة للإدمان
- **IDLE** (خامل): مهام خفيفة / إجراءات إعادة تعيين
- **URGE** (إجهاد): إجراءات طوارئ فورية

### 2. **نظام التسجيل الديناميكي**

كل عنصر يحصل على درجة ديناميكية تأخذ في الاعتبار:

- مطابقة الحالة الأساسية
- عقوبة الانتظار (cooldown)
- عقوبة التكرار (آخر 10 عناصر)
- مكافأة توافق الوقت
- مكافأة مطابقة الشدة
- وزن سجل النجاح

### 3. **نظام مضاد للإدمان**

يتم التفعيل عند:

- تكرار الإجراءات منخفضة القيمة
- التبديل السريع بين العناصر
- الاستخدام في ساعات متأخرة + نمط إنتاجية منخفضة
- تجاوز وقت الخمول الحد الأدنى

**سلسلة الاستجابة**:
1. إجراء حرج مضاد للإدمان
2. إجراء إعادة تعيين جسدي
3. مهمة صغيرة احتياطية (2-5 دقائق)

### 4. **نظام التحقق من التنفيذ**

كل مخرج يتطلب تأكيداً من المستخدم:

- **تم (DONE)**: تحديث الشريط فقط
- **تخطي (SKIP)**: تمديد الانتظار
- **فشل (FAILED)**: تقليل الشدة التالية

## البيانات

### Actions.json

يحتوي على **إجراءات فورية** (0-5 دقائق):

- **الحركة**: امشِ، اجري، اطلع سلالم
- **إعادة التعيين**: تمدد، اشرب مية، اغسل وجهك
- **مضاد الإدمان**: اقفل الموبايل، ابعد عن السرير
- **اللياقة**: ضغط، سكوات، بلانك
- **الروحية**: صلي، اقرأ قرآن، استغفر
- **العقل**: اكتب فكرة، اقرأ، خطط يومك
- **الإنتاجية**: ابدأ فوراً، نظف مكتبك، رتب سريرك

### Missions.json

يحتوي على **مهام منظمة** (10-60 دقيقة):

- **الدراسة**: مذاكرة مركزة، حل مسائل
- **الإنتاجية**: تنظيم يومك، تنظيم الملفات
- **العقل**: جلسة قراءة، كتابة
- **اللياقة**: تمرين جسم كامل
- **الحياة**: تنظيف شامل، تنظيم المكان

## الواجهة

### الزر الأساسي

**"توليد التالي"** - الزر الوحيد الضروري لبدء النظام

### الأوضاع الثانوية

- **وضع الطوارئ**: تفعيل فوري لإجراء مضاد للإدمان
- **المهام فقط**: تصفية لعرض المهام فقط
- **الإجراءات فقط**: تصفية لعرض الإجراءات فقط

### عرض المعلومات

- **العنوان**: اسم الإجراء/المهمة
- **الهدف**: ما تحاول تحقيقه
- **الحالة**: حالتك السلوكية الحالية
- **شرط النجاح**: كيفية معرفة أنك انتهيت

### الإحصائيات

- **السجل**: عدد العناصر المنفذة
- **الشريط**: الإجراءات المتتالية الناجحة
- **الوقت**: الوقت الحالي

## قواعد النظام

### قواعد صارمة

- ✓ لا تكرار العنصر نفسه بالتتابع
- ✓ تجنب نفس الفئة أكثر من مرتين متتاليتين
- ✓ احترام فترات الانتظار (cooldown)
- ✓ تفعيل وضع الطوارئ عند الحالات الحرجة

### القيود

- ✓ نظام بدون اتصال بالإنترنت (Offline-first)
- ✓ لا توجد متطلبات خارجية
- ✓ لا توجد بيانات حساسة
- ✓ لا يوجد إعادة تحديث لا نهائية

## سيناريوهات الاستخدام

### السيناريو 1: حالة التشتت

```
المستخدم: مشتت ويتبديل بسرعة بين التطبيقات
النظام: يكتشف الحالة → يفعل إجراء مضاد للإدمان
النتيجة: إعادة تعيين جسدي فوري (امشِ، اشرب مية)
```

### السيناريو 2: حالة الخمول

```
المستخدم: لم يكن نشطاً لفترة
النظام: يكتشف الخمول → يعين مهمة خفيفة
النتيجة: بدء الزخم (خطط يومك، نظم ملفاتك)
```

### السيناريو 3: حالة التركيز

```
المستخدم: مركز ومنتج
النظام: يكتشف التركيز → يعين مهمة عميقة
النتيجة: تضخيم الإنتاجية (مذاكرة مركزة، حل مسائل)
```

## التثبيت والاستخدام

### المتطلبات

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- JavaScript مفعل
- localStorage متاح

### البدء السريع

1. افتح `index.html` في المتصفح
2. اضغط "توليد التالي"
3. اتبع الإجراء/المهمة المقترحة
4. أكد النتيجة (تم/تخطي/فشل)
5. كرر

### التطوير المحلي

```bash
# استنساخ المستودع
git clone https://github.com/abdoemara175/ETIJAH175.git
cd ETIJAH175

# فتح في المتصفح
open index.html
# أو
python3 -m http.server 8000
# ثم زيارة http://localhost:8000
```
